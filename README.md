## Overview
This project analyzes a Netflix dataset using SQL to uncover trends, patterns, and insights into the platformâ€™s content. By executing SQL queries, it explores key aspects such as content categories, release trends, and regional availability. 
The aim is to understand how Netflix's content library has evolved over time and which types of content are most prevalent.

## Objectives:
- Analyze the distribution of content types (movies vs TV shows).
- Identify the most common ratings for movies and TV shows.
- List and analyze content based on release years, countries, and durations.
- Explore and categorize content based on specific criteria and keywords.
